# mantekio/.github

Organization-level GitHub configuration for **ManTek Technologies**: the org
[profile README](profile/README.md) that appears on the organization's page,
plus any shared community-health files for our public repositories.

ManTek Technologies builds and runs media-grade WordPress and publishing
platforms, with WordPress + AWS at scale. See [mantek.io](https://www.mantek.io)
and our open-source work across the [organization's repositories](https://github.com/mantekio).
